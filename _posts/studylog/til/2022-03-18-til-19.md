- Hook 은 class 안에서 동작하지 않음

  - class 가 아니어도 React를 사용할 수 있게 해주는 것
  - 일반 함수에서도 사용할 수 없음

- 앞에 use가 붙는것은 다 Hook 함수

- 최상위 레벨에서만 호출 가능

  - 커스텀 훅에서는 제외

- props: properties(속성)

  - 단어 뜻 그대로 컴포넌드의 속성값
  - 부모 컴포넌트로붜 전달 받은 데이터를 지니고 있는 개체

- state : 상태 (읽기 전용)
  - 컴포넌트 내부에서 가지고 있는 컴포넌트의 상태값
  - 구조분해 할당 문법으로 사용
  - 상태 변화는 set함수로 사용
