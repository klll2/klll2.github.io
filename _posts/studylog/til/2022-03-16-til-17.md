* Conflict 의 뜻, 충돌이 나는 이유

* 충돌의 해결방법  
  * 로컬의 master 브랜치에 가서 깃헙 mater의 내용을 pull
  * 작업한 브랜치로 가서 `git merge master` 하여 pull
  * vscode를 이용하여 수정 가능
  * Current Change, Incoming Change 등등 클릭해 주거나 직접 수정

* 수정후 커밋 기록을 남기고 재 push
  * 다시 Pull Request

* 충돌 방지를 위해 수시로 pull
