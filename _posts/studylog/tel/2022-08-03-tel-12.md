# TypeScript

---

- SITUATION :

  - VSCode 상에서 Vue 작업을 할때 `template` 태그에서 다음과 같은 에러 메시지가 출력
    ![img](/assets/img/tel/2022-08-03/vue.png/)

- REASON :

  - 템플릿의 TypeScript intellisense 를 활성화 시켜줘야한다고 한다.

- SOLUTION :

  - `jsconfig.json` 에서 다음과 같은 구문을 추가한다. `"jsx": "preserve"`

```js
// file: "jsconfig.json"
  "compilerOptions": {
    "jsx": "preserve",
  }
```
