# CSS

---

- SITUATION :

  - HTML로 마크업하고 CSS로 꾸민 웹페이지가 로딩이 잘 될때가 있고 다 깨질때도 있음

- REASON :

  - 검색으로 알아본 결과 `@import` 의 CSS 로딩 방식은 속도측면에서 좋지 않음

  - 메인 CSS 말고는 쪼개놓은 모든 CSS가 `@import` 로 묶여 묶여 있었음

- SOLUTION :

  - `@import` 를 없애고 각각의 CSS 파일들을 `<link>` 로 HTML HEAD 부분에 넣어줌

  - 이후로 CSS 로딩이 잘 되었음.
