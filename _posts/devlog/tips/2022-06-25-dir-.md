# 타입스크립트 절대경로 지정하기  
---

요즘 리액트에서 타입스크립트를 많이 쓰게 되면서 예전에 자바스크립트에서  
절대경로 지정을 썼던 것이 생각나 타입스크립트도 있나 찾아보았습니다.  

그랬더니, 역시나 존재 합니다!  

`tsconfig.json` 내에 이것을 추가 하면 됩니다!  
자바스크립트때랑 똑같은거 같습니다.  

```java
// file: "tsconfig.json"
{
  "compilerOptions": {
    ...
    ... 다른 내용들
    ...
    "baseUrl": ".", 
    "paths": {
      "@/*": [  // 절대경로를 어떻게 할건지 설정 가능
        "src/*" // 절대경로가 위치하는 곳
      ]
    }
  }
}
```
